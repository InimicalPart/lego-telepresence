# Custom Modules - Written by Inimi

- ``inimi-poweredup``
 - **A modified version of ``node-poweredup`` made by [@nathankellenicki](https://github.com/nathankellenicki) on GitHub.**
 - This modified version uses the [``node-ble``](https://www.npmjs.com/package/node-ble) package (but modified) instead of [``@abandonware/noble``](https://www.npmjs.com/package/@abandonware/noble) which wasn't stable enough for usage.

- ``inimi-ble``
 - **A modified version of ``node-ble`` made by [@chrvadala](https://github.com/chrvadala) on GitHub.**
 - This modified version allows the user to get the server IDs before even connecting to the device.
